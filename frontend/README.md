# EXPENSE-TRACKER:
 
 Simple expense tracker using MERN.

 # FEATURES:
    Basic features include -
    Login and SignUp 
    Income/Expense Transaction
    EXpense and income Forms
    Delete Transactions
    Chart
    Dashboard
    Responsive UI 

# Technology used:
  - MongoDB for database.
  - Node.js and Express for the backend.
  - React for Frontend.
